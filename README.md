# Mum Packaged Drinking Water Website

A premium, responsive website for Mum Packaged Drinking Water - a new business launching in tier 2 and tier 3 cities across India.

## Features

- 6-page responsive website (Home, About, Process, Products, Certifications, Contact)
- TopBar with contact info (phone, email, WhatsApp)
- "Who We Serve" section highlighting customer segments
- Product showcase with 4 bottle sizes
- 6-stage purification process display
- FSSAI & ISI certification highlights
- Bulk enquiry contact form

## Tech Stack

- React + TypeScript
- Tailwind CSS + Shadcn/UI
- Express.js backend
- Wouter routing
- React Hook Form + Zod validation
- TanStack Query

## Brand

- **Brand Name**: Mum
- **Tagline**: Pure • Safe • Refreshing
- **Contact**: wecare@mumdrinks.com
- **Target Market**: Tier 2 and 3 cities in India

## Getting Started

```bash
npm install
npm run dev
```

The app runs on http://localhost:5000
